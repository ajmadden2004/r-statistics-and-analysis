# r-statistics-and-analysis

This repository contains a collection of R scripts used for short examples and statistical analyses.

- Script_0115.R: R script with exploratory data analysis and example plots.
- Script_0120.R: R script focused on data cleaning and transformation steps.
- Script_0122.R: R script demonstrating basic statistical modeling (e.g., regressions).
- Script_0127.R: R script for data visualization and summary reporting.
- Script_0129.R: R script containing helper functions and reusable analysis snippets.
- Script_0203.R: R script with final analyses and consolidated results.
